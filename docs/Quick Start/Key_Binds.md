## Key Binds

Some options in the menu have key binds to speed up your story planning. Below is a comprehensive list of all the key binds on the menu bar.

*On macOS, use the Command key (&#8984;) instead of Ctrl, and Option (&#8997;) instead of Alt.*

### File Commands

| Windows | macOS | Action |
|---------|-------|--------|
| Ctrl+O | &#8984;O | Open or create a story |
| Ctrl+S | &#8984;S | Save story |
| Ctrl+Shift+S | &#8679;&#8984;S | Save as |
| Ctrl+B | &#8984;B | Create backup |
| Ctrl+Q | &#8984;Q | Exit / Quit |

### Add Story Element Commands

These shortcuts are available when the story explorer is visible.

| Windows | macOS | Action |
|---------|-------|--------|
| Alt+F | &#8997;F | Add Folder |
| Alt+A | &#8997;A | Add Section |
| Alt+P | &#8997;P | Add Problem |
| Alt+C | &#8997;C | Add Character |
| Alt+L | &#8997;L | Add Setting |
| Alt+S | &#8997;S | Add Scene |
| Alt+W | &#8997;W | Add Website |
| Alt+N | &#8997;N | Add Notes |
| Alt+B | &#8997;B | Add StoryWorld |
| Delete | Delete | Move selected element to Trash |

### Tools Commands

| Windows | macOS | Action |
|---------|-------|--------|
| Ctrl+N | &#8984;N | Open Narrative Editor |
| Ctrl+K | &#8984;K | Key Questions |
| Ctrl+M | &#8984;M | Master Plots |
| Ctrl+D | &#8984;D | Dramatic Situations |
| Ctrl+L | &#8984;L | Stock Scenes |

### Report Commands

| Windows | macOS | Action |
|---------|-------|--------|
| Ctrl+P | &#8984;P | Print Reports (Windows) / PDF Reports (macOS) |
| Ctrl+Shift+P | &#8679;&#8984;P | PDF Reports |
| Ctrl+R | &#8984;R | Scrivener Reports |

*Note: Print Reports is only available on Windows. On macOS, Ctrl+P opens PDF Reports instead.*

### Other

| Windows | macOS | Action |
|---------|-------|--------|
| Ctrl+, | &#8984;, | Preferences |
| F1 | F1 | Help |
